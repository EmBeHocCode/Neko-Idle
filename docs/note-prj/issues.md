# Issues

- Chưa có bug kỹ thuật trong source game vì dự án chưa triển khai mã nguồn chính thức.

- Đã bổ sung bản tương ứng vào `docs/project-main/` để AI mới có thể đọc đúng vị trí chuẩn.
